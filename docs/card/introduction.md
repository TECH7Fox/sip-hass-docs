# SIP HASS Cards



## Goal

The main goal of the card is to act as a intercom between dashboards like wall-mounted tablets and desktops, and SIP devices like doorbells and deskphones.

## Features

- Audio visualiser
- Auto answer
- Auto call
- DTMF
- Mute mic/video
- Ringtone and Backtone
